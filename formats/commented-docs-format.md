## Commented Docs Format

This file contains the format for the commented docs for all over script and source code files that I ([rdofficial](https://github.com/rdofficial/)) ever create.

### Formats

The docs format is written below in list :
* __bash/shell script__ :
```
# ----
# script_name [bash / shell script] {OPTIONAL}
#
# {DEFINITION + BASIC INTRODUCTION + BASIC APPLICATION + BASIC ALGORITHM FOR THE SCRIPT}
#
# {DEPENDENICES - IF ANY OTHERWISE LEAVE IT, IT'S OPTIONAL }
#
# Author : Rishav Das (https://github.com/rdofficial/)
# Created on : -
#
# Last modified by : -
# Last modified on : -
#
# Changed made in last modification :
# {CHANGES TO BE SPECIFIED IN NUMERIC ORDER, IF ANY OTHERWISE LEAVE THE SECTION BLANK}
#
# Authors contributed to this script (Add your name below if you have contributed) :
# 1. Rishav Das (github:https://github.com/rdofficial/, email:rdofficial192@gmail.com)
#
# ----
```
